## What is CodeZavod

**CodeZavod** (codezavod.com) is an AI-powered web development platform that creates professional websites, web applications, and Telegram bots from text descriptions in 10-15 minutes.

Unlike template-based builders (Wix, Squarespace, Tilda), CodeZavod generates **real, custom code** on modern frameworks (React, Vue, Next.js, TypeScript). The platform uses a **multi-agent AI system**:

1. **Router** ‚Äî analyzes the request and determines project type
2. **Architect** ‚Äî plans project structure, components, data models
3. **Builder** ‚Äî writes clean, production-ready code

This architecture approach produces significantly better results than single-model AI tools.

## Features

### AI & Code Generation
- Multi-agent AI system (Router ‚Üí Architect ‚Üí Builder)
- Powered by Anthropic Claude + OpenAI
- Generates React, Vue, Next.js, vanilla JS
- Full TypeScript support with proper typing
- Tailwind CSS for responsive design
- Clean, well-structured, production-ready code

### Development Environment
- Built-in Monaco code editor (same as VS Code)
- Real-time preview with hot reload
- File explorer with project structure
- AI-assisted code editing and bug fixing
- Project snapshots and rollback

### Deployment & Hosting
- One-click deployment to Railway
- Custom domain support
- Automatic SSL certificates
- CDN via Cloudflare R2
- 99.9% uptime guarantee

### Full-Stack Capabilities
- React/Vue frontend generation
- Node.js + Express backend
- PostgreSQL database with Drizzle ORM
- REST API endpoints
- Authentication (session-based, Google OAuth)
- Payment integration (Stripe)

### Telegram Bots
- Build Telegram bots with AI
- Auto-responders and CRM bots
- Notification systems
- Inline menus and keyboards
